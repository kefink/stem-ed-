## 💻 How to Add VR Experiences

Edit `page.tsx` and add to `subjectCategories` array:

```typescript
{
  id: "new-subject",
  name: "Subject Name",
  icon: "🎯",
  color: "from-blue-500 to-indigo-600",
  description: "Description here...",
  experimentCount: 20,
  highlights: [
    "Feature 1",
    "Feature 2",
    "Feature 3"
  ],
  gradeLevels: ["Grade 7", "Grade 8", "Grade 9"] // Optional for CBC
}
```
